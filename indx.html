O que é RGB?

RGB é um modelo de cores aditivo baseado na combinação de três cores primárias: Vermelho (Red), Verde (Green) e Azul (Blue). Esse modelo é usado em dispositivos eletrônicos, como monitores e televisões.

O Modelo Aditivo

No modelo aditivo, a luz emitida por cada pixel combina-se para formar outras cores. Os pixels das telas possuem subpixels de RGB, e a intensidade de cada cor determina a cor final exibida.

  • Vermelho + Verde = Amarelo
  • Vermelho + Azul = Magenta
  • Verde + Azul = Ciano
  • Vermelho + Verde + Azul = Branco

O que é CMYK?

CMYK é um modelo de cores subtrativo utilizado em impressão. Ele se baseia na mistura dos pigmentos Ciano (Cyan), Magenta (Magenta), Amarelo (Yellow) e Preto (Key). A cor preta é usada para gerar contraste e reduzir desperdício de tinta.

O Modelo Subtrativo

No modelo subtrativo, a luz refletida sobre os pigmentos determina a cor percebida. A combinação do ciano, magenta e amarelo pode formar diversas cores, mas a adição do preto melhora o contraste.

  • Ciano + Magenta = Azul
  • Ciano + Amarelo = Verde
  • Magenta + Amarelo = Vermelho
  • Ciano + Magenta + Amarelo + Preto = Preto Profundo
